# XHorizon

[![Status][status-badge]][status-link] [![Discord][discord-badge]][discord-link] [![Website][website-badge]][website-link]

XHorizon is a Discord-like social media application currently under development. It aims to provide a flexible, safe, and modern chat platform with real-time communication, server management, and content moderation.

</div>

---

## Features

- User registration and login with password hashing
- Account recovery via email or SMS
- Age verification: 16+ to register, 18+ to access NSFW content
- Privacy settings and account management
- Two-factor authentication (2FA)
- Real-time chat 
- Create and join servers (guilds) and channels (text, voice, media, announcement, forum, calendar, lists, docs)
- Send, edit, and delete messages
- Direct messages between users
- User roles and permissions (admin, moderator, member)
- Emoji picker and reactions
- File and image uploads
- Moderation tools: kick, ban, content reporting
- Anti-CSAM, anti-scam, and grooming detection system
- Option to block NSFW content
- Reports and escalation to human moderation when necessary
